Sobre Mim

Sou Guilherme dos Santos Mendonça Enéas, Engenheiro de Software com foco no desenvolvimento backend e arquitetura de sistemas distribuídos. Possuo mais de 1 ano e meio de experiência prática em Java e SpringBoot, com foco em microsserviços e APIs RESTful, sempre seguindo os princípios de Clean Code, SOLID e práticas ágeis.

Minha trajetória inclui a construção de soluções escaláveis e de alta performance, com domínio em tecnologias como Docker, MongoDB, Apache Kafka, CI/CD, JUnit e Mockito. Tenho sólida experiência no design e desenvolvimento de APIs, testes automatizados, versionamento com Git, documentação no GitHub e uso de padrões como DDD e TDD.

Atuei também como freelancer na plataforma Workana, prezando por código limpo, organização e entrega de valor ao cliente. Em 2 projetos reais, destaques para o desenvolvimento e melhoria de uma API RESTful projetada para gerenciar mensagens de comunicação interna com suporte integrado para notificação por e-mail, incluindo documentação completa, mapeamento DTO-Entity, tratamento de exceções, suporte a Docker e testes unitários. E o outro foi uma comunicação interna com envio de e-mails, onde implementei cadastro de usuários, agendamento de tarefas, notificações automatizadas.

Atualmente estou cursando Análise e Desenvolvimento de Sistemas pela UniCesumar, pretendo fazer uma pós de Software Architecture e uma MBA em AI, Data Science e Big Data. Sigo em constante evolução rumo ao objetivo de assumir papéis estratégicos em projetos de alto impacto, contribuindo na construção de soluções robustas, modernas e sustentáveis.

Paralelamente, estou expandindo minha atuação para o desenvolvimento FullStack, estudando Angular, React, React Native e a linguagem Go, além de aprofundar meus conhecimentos em Cloud Computing, DevOps, Inteligência Artificial.

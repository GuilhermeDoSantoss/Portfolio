Sobre Mim

Sou Guilherme dos Santos Mendonça Enéas, Software Engineer com foco em desenvolvimento backend em Java e arquitetura de sistemas distribuídos. Tenho mais de 2 anos de experiência prática no desenvolvimento de APIs RESTful, microsserviços e soluções escaláveis, aplicando Clean Code, SOLID, DDD e TDD em projetos reais.

Minha trajetória inclui atuação em freelancer pela Workana e no desenvolvimento de aplicações voltadas para diferentes segmentos (educacional, esportivo, estético), com destaque para sistemas de comunicação interna com envio de e-mails, agendamento de tarefas e notificações automatizadas, além de APIs documentadas, testadas e integradas a pipelines CI/CD com Docker.

Tenho experiência consolidada em Java, Spring Boot, Kafka, RabbitMQ, MongoDB, PostgreSQL, Docker, GitHub Actions e testes automatizados (JUnit5/Mockito). Minha rotina de trabalho é sempre guiada por boas práticas de engenharia, padrões arquiteturais modernos e foco em entregas de valor.

Atualmente sou graduando em Análise e Desenvolvimento de Sistemas pela UniCesumar, e sigo me preparando para uma pós-graduação em Software Architecture e um mestrado em Ciência da Computação.

Paralelamente, estou expandindo minha atuação para o desenvolvimento FullStack, estudando Angular, React, React Native e Go, além de aprofundar meus conhecimentos em Cloud Computing (AWS/ Azure/Google Cloud), DevOps e IA.

Meu objetivo é assumir papéis estratégicos como Backend Developer Java ou Fullstack Developer, contribuindo em projetos de alto impacto que envolvam arquitetura, nuvem e sistemas distribuídos, sempre em ambientes ágeis, colaborativos e orientados à excelência técnica.
